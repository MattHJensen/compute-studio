COMPmodels.org is an open-source utility for sharing computational models.

Features:

  • Share public models and simulations for free.
  • Pay for your own compute at cost.
  • Sponsor compute for others if you'd like.

Under development:

  • Non-tabular output.

Planned:

  • REST API to run simulations and build GUI simulation pages.

Please contribute code or suggestions on GitHub. We also have a COMP developer community chat room.
